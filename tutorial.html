Mini Lesson 1 — Variable Basics

A variable is a name that points to a value your program wants to remember and use later.

Not completed

Key Idea

In Python, score = 10 means the variable named score is storing the value 10. The equals sign here is used for assignment, not for asking a question.

Example

name = "Maya"
age = 14
score = 10

Mini Lesson 2 — Common Data Types

Python treats numbers, text, and true-or-false values as different kinds of data.

Not completed

Key Idea

Four beginner data types show up often: integers for whole numbers, floats for decimals, strings for text, and booleans for true-or-false values.

Example

lives = 3        # integer
price = 3.14     # float
city = "Rome"    # string
ready = False    # boolean

Mini Lesson 3 — Naming Variables Clearly

Good variable names help readers understand the purpose of the value immediately.

Not completed

Key Idea

Python variable names cannot start with a number and cannot include spaces. A common style is to use lowercase words joined with underscores.

Example

student_name = "Ava"   # good
2name = "Ava"          # not valid
first name = "Ava"     # not valid

Mini Lesson 4 — Reading Simple Code

Students should be able to look at a short line and explain the stored value and the type.

Not completed

Key Idea

When you read code, identify the variable name first, then the value, then the type. This keeps students from guessing too fast.

Example

score = 8
city = "Rome"

Reading Strategy

Ask three short questions: What is the variable called? What value is stored? What kind of value is it?

Quick Check

Study Tips

  • Have students say the variable name and value out loud before naming the data type.
  • Use very short examples first so students focus on meaning instead of syntax overload.
  • Compare two lines side by side so students can notice the difference between numbers and text.
  • Encourage students to rewrite unclear names into better ones such as student_name or quiz_score.
