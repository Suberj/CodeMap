Python Installation Guide

Follow these steps in order to set up Python, verify your installation, and get ready to start running code locally.

Step 1: Check if Python is already installed

Open your terminal and run the following command to see if Python is already available on your machine.

python --version
Terminal screenshot placeholder: expected output example goes here.

Step 2: Install Python

If Python is not installed, download it from the official website and complete the installer steps.

External download link: python.org/downloads

Step 3: Verify pip

After installation, confirm that the package installer is also available.

pip --version

Step 4: Create a project folder

Make a clean folder for your coursework or practice files.

mkdir my-python-project cd my-python-project

Step 5: Run your first test file

Create a file called hello.py and run it to confirm the environment works.

python hello.py
